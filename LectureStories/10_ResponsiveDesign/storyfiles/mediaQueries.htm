Media Queries

Media Queries will let you:

  • adjust your site design to the media
  • identify needed breakpoints
  • Improve navigation on small screens

We start with this working file: Kadlec ch3.1.html which has the previous components plus a header and a footer.

If the window is very wide, the text passages are too wide to read comfortably.

If the window is narrow, header and footer items move. The text passages get squeezed to two or three words per line.

This is not what you would see on a mobile phone. It will look like the original page but zoomed out. The text will be tiny.
