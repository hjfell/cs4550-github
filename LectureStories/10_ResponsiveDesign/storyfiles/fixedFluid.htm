Mixing Fixed and Fluid

To keep advertisers happy, we want the aside column fixed at 300px.

Fixed width for the aside and 63.125% for the main column does not work well if the screen is less than 960 pixels.

Table-like Layouts

Tables let you mix fixed an fluid layouts but they are considered verboten by CSS standards.
Table values for CSS define visual layout not content.

CSS display property gives similar control.

See w3Schools CSS display Property


.main {
    display: table-cell;
    padding-right: 2.5316456%; /* 24px / 948px */
}

aside {
    display: table-cell;
    width: 300px;
}
