Make it Flexible

Kadlec ch2.3.html is the same as Kadlecch2-start.html but it uses the css file on the next slide.


aside {
    float: right;
    width: 31.6455696%; /* 300/948 */
}
.main {
    float: left;
    width: 65.8227848%; /* 624/948 */
}

Now remove the container fixed width.


#container {
    margin: auto 0;
    overflow: hidden;
    width: 95%;			// because it works
    padding: .625em 1.0548523% 1.5em; /* 10px/16px, 10px/948, 24px/16px */
    		// ems for the top and bottom (determined by font size)
    		// and percentages for the left and right
}

Images

Fixed-width images are too small on a large display and too wide on small displays.

Tell the image to fill the width, e.g. of the aside.


    aside img,
    .main img,
    .slats img{
        width: 100%;
        	// If you set both width and height, the image will not adjust proportionately.        
        max-width: 100%;
			// This keeps the image from spilling over or getting cut off.
    }
