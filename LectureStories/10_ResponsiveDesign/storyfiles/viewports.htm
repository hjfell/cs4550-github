Viewports

For a desktop browser, the viewport is the visible area of the browser
and a pixel is a pixel.

On phones there are device pixels and CSS pixels.

CSS pixels don't always line up with device pixels

  • On a Retina display, one CSS pixel is two device pixels.
  • If the user zooms to 200%, the CSS pixels double their height and width.
  • The number of device pixels doesn't change.
  • The number of CSS pixels that can be viewed inthe browser does change.

To make things work better (or at least differently) on phones, add this to the <head> of your html file. 

	<meta name="viewport" content="width=device-width" />

Avoid these:

  • <meta name="viewport" content="width=320px" />
  • <meta name="viewport" content="height=anything_at_all" />
  • <meta name="viewport" content="userscalable=no" />
  • <meta name="viewport" content="minimum-scale=anything_at_all" />
  • <meta name="viewport" content="maximum-scale=anything_at_all" />
