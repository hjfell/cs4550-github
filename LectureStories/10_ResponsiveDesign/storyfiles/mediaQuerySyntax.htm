Media Query Syntax

Instead of the meta clause, we can now use rules like this in our CSS.


	@viewport {
		width:device-width;
	}

In general


	@media [not|only] type [and] (expr) {
		rules
	}

You can find out what is true and load styles accordingly.

You can also use links to appropriate stylesheets.


	<link rel="stylesheet" href="print.css" media="print" />

A media query has

  • media types to target
  • expressions to test for certain features
  • logical keywords
  • Rules that adjust the display
