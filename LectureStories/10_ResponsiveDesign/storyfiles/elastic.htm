Elastic Layouts

Constraints are determined by type size using the em.

One em is approximately the font size in pizels, e.g. ~24px for a 24px font.

Elastic layouts provide strong typographic control.

A good line length is between 45 and 70 characters. You can make a container that is 55em wide.

If a viewer changes the font size, elements will be scaled in proportion to the new font size.

55em at 16px font size is 880px and won't fit on a tablet. -- The horizontal scrollbar appears.

Can be really unpredictable if the viewer makes the font a larger size.
