Fixed Width

The layout is constrained to a fixed pixel measurement.

The most common width is 960px.

  • Assumes 1024px for the display width.
  • The edges of the browser use some of those pixels so ~975 to ~985 are left.
  • 960 fits and has lots of divisors.
  • That allows for many grid options that fit standard ads sizes.

Fixed width lets you really control the layout.

Problems arise when people

  • don't use the full width of their display for their browser
  • have browser plugins that take up extra space
  • use tablets or cell phones
  • have super-wide displays
