HTML5 Audio <audio>

Provide Alternate Sources and Fall Back for Older Browsers


<audio controls autoplay>
    <source src="../sounds/Bach_ Invention_10_in_G_Major.mp3">
    <source src="../sounds/Bach_ Invention_10_in_G_Major.ogg">
    <!-- fallback -->
       <p>Your browser does not support HTML5 video.</p>
</audio>

The <source> tag accepts attributes for MIME type and codecs as well.

Control the volume with JavaScript


<script>
myVid=document.getElementById("player1");
function getVolume()
  { 
  alert(myVid.volume);
  } 
function setHalfVolume()
  { 
  myVid.volume=0.2;
  } 
function setFullVolume()
  { 
  myVid.volume=1.0;
  } 
</script>
