What is MySQL

MySQL is the world’s most popular open-source RDMNS.

Relational Database Management System includes:
mysqld runs and manages the databases
mysql* gives you an interface to the server
other utilities

* We will actually use mysqli, the MySQL improved extension that was developed to take advantage of new features found in MySQL systems versions 4.1.3 and newer.

PHP has good support for MySQL.

MySQL is used by NASA and the US Census Bureau.
