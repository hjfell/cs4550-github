Debugging PHP

To debug a PHP script

  1. Make sure you’re always running PHP scripts through a URL! (i.e. via http://something)
  2. Know what version of PHP you’re running. Run a phpinfo.php script or reference the PHP_VERSION constant
  3. Make sure display _errors is on. This is a basic PHP configuration setting. Check by executing phpinfo( ).
  4. Look for errors in your browser.
  5. Check the HTML source code. Sometimes the PHP error message can be hidden there!
  6. Trust the error message.
  7. Take a break!
