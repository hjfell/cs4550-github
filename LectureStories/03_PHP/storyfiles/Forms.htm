Forms

<form action="script.php" method="post">
</form>

action dictates to which page the form data will be sent

method dictates how the data is sent to the handling page,
the HTTP (HyperText Transfer Protocol) method to be used

get sends the submitted data to the receiving page as a series of name-value pairs appended to the URL. For example,
http://www.example.com/script.php?name=Homer&gender=M&age=35

• resulting page can be bookmarked in the user’s Web browser (since it’s a complete URL)
• can click Back in your Web browser to return to a GET page, or reload it without problems
• there is a limit in how much data can be transmitted via GET
• this method is less secure (since the data is visible)
• used for requesting information, like a particular record from a database
• or the results of a search

post is used when an action is expected
• the updating of a database record
• the sending of an email

All different kinds of input can be in the form.
