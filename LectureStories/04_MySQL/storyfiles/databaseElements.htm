Naming Database Elements

Identifiers

  • Should contain only letters numbers and underscore
  • Should not be the same as existing keywords
  • Should be treated as case-sensitive (depends on OS but safer this way)
  • Should be at most 64 characters long (64 bytes really)
  • Should be unique within their realm
  • Cannot have two columns with the same name
  • Two tables cannot have the same name
  • Two different tables can have columns with the same name

Steps to Setting up a Database

  1. Pick a database name – might be set by your host provider (mysite)
  2. Determine the tables (users)
  3. Determine the column names for each table
