Choosing Column Types

Primary Types

  • Text
  • Numbers
  • Dates and Times

Common variants: http://www.w3schools.com/sql/sql_datatypes.asp
Many types take an optional Length attribute.

To select column types

  1. Identify the primary type
    Column NameType
    user_idnumber
    first_nametext
    last_nametext
    emailtext
    passtext
    registration_datedate/time

  2. Choose the best subtypes
    Column NameType
    user_idMEDIUMINT
    first_nameVARCHAR
    last_nameVARCHAR
    emailVARCHAR
    passCHAR
    registration_dateDATETIME

  3. Set the maximum length for the columns
    Column NameType
    user_idMEDIUMINT
    first_nameVARCHAR(20)
    last_nameVARCHAR(40)
    emailVARCHAR(60)
    passCHAR(40)
    registration_dateDATETIME
