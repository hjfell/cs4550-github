What is MySQL

MySQL is the world's most popular open-source relational database management system
(according to www.mysql.com).

MySQL is used by NASA and the US Census Bureau.

It is commonly used with PHP.

MySQL comes with

  • A database server to store the actual data.
  • Different client applications for interacting with the data
  • Several utilities

We will create a simple table and learn how to interact with MySQL.
