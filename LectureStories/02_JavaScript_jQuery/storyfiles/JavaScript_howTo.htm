Writing JavaScript

JavaScript code can be in separate files or embedded in HTML.
Scripts in HTML must be inserted between <script> and </script> tags.
Scripts can be in the document head or body.
Functions are usually defined in other files, the head, or just before </body>.
Scripts in separate files do not have in the .js file. The HTML include the .js file with a line like this:
<script type="text/javascript" src="myTools.js"></script>

  • document.getElementById(id)
  • document.write("<p>some text</p>");
    Warning: Do not execute document.write after the document has finished loading
  • JavaScript Statements
    Ending statements with semicolon is optional but highly recommended in JavaScript.
    JavaScript is case sensitive and ignores white space.
  • JavaScript Comments

  • JavaScript Numbers are just like numbers should be.
    JavaScript Math object has methods that act on numbers.
  • JavaScript Strings
    • can be wrapped in single quotes or double quotes
    • can contain zero or more characters
    • \ (backslash) is the escape character
    Characters in JavaScript are 16 bits wide (for Unicode 16).
  • JavaScript Variables
    • hold values of expressions
    • names start with letters (or $ or _ but you might avoid these)
    • names are case sensitive
    • don't use reserved words
  • JavaScript Data Types
    • are dynamic
    • happen by assignment
      var name="Jerry", age=30 + 1/2, job='superhero', annual_salary=4300.00;
      document.write(name + " is a " + age + " year old " + job + " who earns $" + annual_salary + " per year.");
    • or can be declared with new
      <script>
      var dogname=new String;
      document.write('<br />');
      document.write('My dog is ' + dogname + '<br />');
      dogname='Tasha';
      document.write('My dog is ' + dogname + '<br />');
      dogname= 6+4;
      document.writeln('My dog is ' + dogname + '<br />');
      </script>
  • JavaScript Operators
    are pretty much like operators you have seen in other languages.
    • Arithmetic Operators:
      +-*/%++--
    • Assignment Operators:
      =+=-=*=/=%=
    • Concatenation: +
    • Comparison Operators:
      =====!=><>=<=
    • Logical Operators
      &&||!
    • Conditional Operator: variablename=(condition)?value1:value2
