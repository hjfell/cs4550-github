What is JavaScript?

JavaScript

  • is a Scripting Language
    i.e. a lightweight programming language
  • can be inserted into HTML pages
  • can be executed by all modern web browsers
  • is executed line by line while the browser reads it
  • came from many languages:
    • Syntax from Java
    • Functions from Scheme
    • Prototypal Inheritance from Self
    • Regular Expression feature from Perl
