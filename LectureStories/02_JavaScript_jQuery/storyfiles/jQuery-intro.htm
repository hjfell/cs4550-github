What is jQuery?

jQuery is a JavaScript library that makes it much easier to use JavaScript on your website.

jQuery usually takes many fewer lines of code than pure JavaScript to accomplish the same tasks.

jQuery also simplifies AJAX (Asynchronous JavaScript and XML) calls and DOM (Document Object Model) manipulation.

jQuery includes

  • HTML/DOM manipulation
  • CSS manipulation
  • HTML event methods
  • Effects and animations
  • AJAX
  • Utilities
  • lots of plugins
  • jQuery UI has useful effects and advanced widgets that are accessible and highly customizable, e.g. accordions, sliders, dialog boxes, and date pickers.

jQuery lacks a mechanism for adding inline code, which leads to leaner, cleaner, and more maintainable code.

Loading jQuery

Include the jQuery library in the <head> of your page.

  • for the current latest version
    (spring 2013)
    <script src="//ajax.googleapis.com/ajax/libs/jquery/1.9.0/jquery.min.js"></script>
  • for the current latest version
    (spring 2014)
    <script src="http://code.jquery.com/jquery-1.10.1.min.js"></script>
    <script src="http://code.jquery.com/jquery-migrate-1.2.1.min.js"></script>
  • for the latest version in the "1" series
    <script type="text/javascript" src="//ajax.googleapis.com/ajax/libs/jquery/1/jquery.min.js"></script>
  • or a local copy that you downloaded, e.g.
    <script src="../jQuery/lib/jquery-1.7.1.min.js"></script>
