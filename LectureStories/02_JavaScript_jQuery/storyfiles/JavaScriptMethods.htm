Some String Methods

  • string.charAt(pos) returns a string
    var name = 'Curly'; var initial = name.charAt(0); // initial is 'C'
  • string.charCodeAt(pos)
    var initial = name.charCodeAt(0); // initial is 67
  • string.charCodeAt(pos)
    var initial = name.charCodeAt(0); // initial is 67
  • string.concat(string...)
    var s = 'C'.concat('a', 't');    // s is 'Cat'
