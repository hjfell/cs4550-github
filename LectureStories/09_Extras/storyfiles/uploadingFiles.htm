Uploading Files

Make and run a phptest.php file with the contents


	<?php phpinfo(); ?>

Check these fields:

DirectiveValueValue Type
file_uploadsOnBoolean
max_input_time60Integer
max_file_uploads20Integer
post_max_size32MInteger
upload_max_filesize32MInteger
upload_tmp_dir/Applications/MAMP/tmp/phpString

You can reset them in your php.ini file.
