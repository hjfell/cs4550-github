 This is an interactive viewer and designer for 2-dimensional Lindenmayer systems.
When you first load it, the fields at the top have the information necessary to generate a Koch Triagle form. Click on "submit" to see the system rendered in the drawing area.
If you click on one of the eight buttons to the right of the drawing area, the information for that system will be entered into the fields and a click on "submit" will draw the new shape.
You can modify the entries in the fields to generate your own shapes. This version of the program does not have error checking so try to enter appropriate information.
  • Start can be a string of (at least one) upper or lowercase letters and the symbols '+' and '-'.
  • Rules start with a letter (upper or lower case) followed by a colon followed by a string of upper and lowercase letters, the symbols '+' and '-' and matched pairs of the symbols '[' and ']'.
  • Try Wikipedia: L-system for more information.
If you like a system you have designed, give it a name and click on the "save" button. When you reload the page, you should see a new button for each system that you saved.
A "delete" button will appear next to each of your creations so you can delete any you don't like. Deleted items will not show up when you reload.
