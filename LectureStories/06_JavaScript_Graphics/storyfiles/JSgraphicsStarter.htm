Getting Started with JavaScript Graphics

Create a drawing canvas on your html page.

<canvas id="my_canvas_id" width=300 height=300 style="border: 1px solid black;"></canvas>

Name the canvas and create a 2d context for it.


		var canvas = document.getElementById("my_canvas_id"); 
		var c = canvas.getContext('2d');

Then do some simple drawing


		c.beginPath(); 	// Start a new path 
		c.moveTo(100, 100); 	// Begin a path at (100,100) 
		c.lineTo(200, 200); 	// Add a line from (100,100) to (200,200) 
		c.lineTo(100, 200); 	// Add a line from (200,200) to (100,200)

		c.fillStyle = "#f00"; 	// Set fill color to red
		c.fill(); 	// Fill a triangular area 
		c.stroke(); 	// Stroke two sides of the triangle
