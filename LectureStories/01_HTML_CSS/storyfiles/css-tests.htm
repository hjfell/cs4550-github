CSS Tests

In this version of the Tire Sizing article, tire-sizing-03, I have moved the inline style attributes to a style declaration in the document head. Here are some of the changes that you should see in the body of the document,

  • <p style="text-align: center;"> has been changed to <p class="center">
    Similar changes appear in <h2> tags.
  • The title style is now set with id=articletitle
  • The author style is now set with id=byline
  • All the tables have style set with class="grid"
  • The cumbersome style atributes that were in the navigation list and its elements are covered by <ul class="hlist">

In tire-sizing-04, the styles have been moved to a css file, bike-article-style.css

This file, RR-style-css-tests.htm, shows some experiments with the main style sheet style.css that Professor Rasala provided for his site.
