CSS Syntax

A CSS rule has two main parts: a selector, and one or more declarations.
A selector is usually the HTML element you want to style.
Each declaration consists of a property and a value.

/*This is a css rule.*/
p
{
/*It has two declarations.*/
color:red;
text-align:center;
}

The id Selector
is used to specify a style for a single, unique element. 

#articletitle 
{
	font-size:300%;
	color:red;
	text-align:center;
	font-weight:bold;
	margin-top: 0.5em;
}

The class Selector
is used to specify a style for a group of elements. 

.inset1 {
    margin-left: 5%;
    margin-right: 5%;
}

A class defined only for images

img.floatRight { 
    float: right; 
    margin: 4px; 
}
