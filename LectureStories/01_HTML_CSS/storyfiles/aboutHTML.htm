What is HTML?

Hyper Text Markup Language is a language for describing web pages.

HTML is a set of markup tags that describe document content

An HTML document (web page) contains HTML tags and plain text.

HTML Tags

  • keywords (tag names) surrounded by angle brackets like <html>.
  • HTML tags normally come in pairs, a start or opening tag and an end or closing tag, like <b> and </b>
  • An end tag looks just like the start tag but has a forward slash before the tag name, e.g.
    <tagname>content</tagname>

HTML Elements

An HTML element is everything between the start tag and the end tag, including the tags:
<p>This is a paragraph.</p>

A browser's job is to interpret the HTML tags to properly display the document.

HTML Attributes

HTML elements can have attributes that provide additional information about an element.

  • Attributes are specified in the start tag
  • Attributes come in name/value pairs like: name="value"
  • Example: <a href="http://www.w3schools.com">This is a link</a>
  • Some others: class, id, style, title

Always quote attribute values.

Lowercase is recommended and required in XHTML.
