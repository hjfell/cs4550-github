SQL Security Measures

  1. Protect the MySQL access information.
    Put mysqli_connect.php outside the Web Directory - if possible.
  2. Do not reveal too much about the database.
    Do not let users see PHP queries or error messages.
  3. Be careful when running queries, especially user submitted queries.
    1. Validate user-supplied data.
    2. Use regular expressions to make sure that the data looks right.
    3. Typecast numbers.
    4. Run user data through mysqli_real_escape_string() as in
      $safe = mysqli_real_escape_string($dbc, data)

If Magic Quotes is enabled on your server, you have to remove the extra slashes:
trim(stripslashes($_POST['first_name'])).

material from Larry Ullman, PHP and MySQL for Dynamic Web Sites, Fourth Edition
