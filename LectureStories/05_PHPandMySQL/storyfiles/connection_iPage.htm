Connecting to MySQL - iPage

Clicking on "Generate Code" will bring up a window like this.

Copy the code and put it in the "mysqli_connect.php" file that you will use on iPage. Note that there is code below the little window so make sure to grab it all. Change the obvious things and be sure to add the "i" in mysqli_connect so that you can run improved PHP to MySQL queries.

<?php
$link = @mysqli_connect('your_iPage_databasename.ipagemysql.com', 'database_name', 'your_password');
if (!$link) {
die('Could not connect: ' . mysql_error());
}
//echo 'Connected successfully';
//mysqli_select_db(yoursite);
// Set the encoding...
mysqli_set_charset($dbc, 'utf8');
?>
