Simple Queries

Simple Queries: INSERT, DELETE, UPDATE, or ALTER

These do not return data. They do return a success value.

Connect to MySQL

// Connect to MySQL.
require ('../../../../../mysqli_connect.php');

Connect to the Database

// Connect to the Database
$q = "USE mysite";
$r = @mysqli_query ($dbc, $q); // Run the query.

Make the Query

// Make the query:
$q = "INSERT INTO users (first_name, last_name, email, pass, registration_date)
VALUES ('$fn', '$ln', '$e', SHA1('$p'), NOW() )";
$r = @mysqli_query ($dbc, $q); // Run the query.

Note the missing ";"s.

material from Larry Ullman, PHP and MySQL for Dynamic Web Sites, Fourth Edition
