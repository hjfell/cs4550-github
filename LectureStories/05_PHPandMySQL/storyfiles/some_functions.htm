Some PHP --> MySQL Functions

Count the number of rows in a response
$num = mysqli_num_rows($r);

Check for a repeated email address - without causing a MySQL error
$e = mysqli_real_escape_string($dbc, trim($_POST['email']));
$q = "SELECT user_id FROM users WHERE email = '$e'";
$r = @mysqli_query ($dbc, $q); // Run the query.
$n = mysqli_num_rows($r);
if ($n > 0) {
$errors[] = "The email address $e is already in use.";
}

Count the number of rows affected by INSERT, UPDATE, or DELETE
$num = mysqli_affected_rows($dbc);

material from Larry Ullman, PHP and MySQL for Dynamic Web Sites, Fourth Edition
