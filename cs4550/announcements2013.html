Thu Mar 01, 2013

It is important for Karishma and me to see your php code (and html, js, css) when grading your reservation systems. Please create a page with links to the source code for all your files and a brief description of what the file does. Files with extension .css or .js will come up as source automatically but html and php files will not

One way to do this is to create a file called myFile_source.php for each php file with this content and then link to these. Please do the same for html files.


<!DOCTYPE html>
<html lang="en">
<head>
	<meta charset="utf-8" />
	<title>myFile.php Source</title>
</head>
<body>
	<h1 class="center">myFile.php Source</h1>
	<?php show_source("<path_to_myFile.php>/myFile.php") ?>
</body>
</html>
